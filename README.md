# PinyinTextView
支持汉语拼音的自定义View
* 支持汉字拼音、汉字、拼音三种文本切换
* 支持首行缩进
* 支持自定义颜色：文本颜色、拼音颜色、下划线颜色
* 支持自定义间距：字间距、行间距、拼音汉字间距、下划线与文字间距
* 支持下划线：两种style、自定义颜色、自定义宽度
* 支持手动换行'\n'
